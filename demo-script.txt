PART 1 — INTRODUCTION  (about 1 minute)
===========================================================================

[Face camera or voice over slides/diagram]

"Hi, my name is Xinyue Zhang, and this is my CS 310 final project:
Animal Rescue Hub — a cloud-native web service that helps small animal
rescue organizations increase visibility and connect adoptable animals
with potential adopters.

I was inspired by Paws and Claws Cat Rescue in Evanston, Illinois,
where I adopted my own cat.

The application has two main user roles:
  - Rescue STAFF who upload animals for adoption
  - VISITORS who browse animals and submit adoption applications

The entire backend runs on AWS."


===========================================================================
PART 2 — ARCHITECTURE OVERVIEW  (about 1.5 minutes)
===========================================================================

[Show or draw a diagram with these components]

"Let me walk through the architecture.

The server is a Node.js / Express web service.
It connects to four AWS services:

  1. Amazon S3 — stores pet photos uploaded by rescue staff
  2. Amazon RDS (MySQL) — stores pet listings, subscribers, and applications
  3. AWS Rekognition — automatically detects the animal species from a photo
  4. Amazon SES — sends email notifications

The server also calls an external third-party API:
  5. The Cat API — provides live cat breed information

The client is a React single-page app served directly by Express,
plus a Python command-line client for testing.

There are three non-trivial server-side operations, which I'll demonstrate now."

--- NON-TRIVIAL #1: External API Call ---  (~1 min)

[Click "Cat Breeds" tab in the browser]

"The first non-trivial operation is GET /external_pets.

When you load this tab, the server makes a live HTTP request to
The Cat API, retrieves breed data, transforms it into our schema,
and sends it back to the browser.

You can see each breed's photo, lifespan, weight, description,
temperament tags, and intelligence/affection/energy ratings —
all fetched in real time. No data is stored in our database.

I can change the number of results here and refresh to get a new set."

[Change dropdown to 16, click Refresh — show new cats loading]
