Hi, my name is Xinyue Zhang, and this is my CS 310 final project — Animal Rescue Hub. I'm a first-year CS graduate student, and this is my first time building a full cloud-native application from scratch.

I built this app because I adopted my own cat from a small rescue called Paws and Claws in Evanston, and I noticed that small rescues often don't have good tools for getting visibility online or managing adoptions. So the idea behind this project is to build a cloud service that helps them do exactly that.

The way it works is — a rescue staff member uploads a photo of an animal, the server automatically figures out what species it is using AWS Rekognition, saves everything to the database, and sends an email to all subscribers to let them know a new animal is available. Visitors can browse the animals, and when they find one they want to adopt, they submit an application and get a confirmation email back.

There are three non-trivial server-side operations, which I'll walk through in the demo. But first, let me quickly go over the architecture.

In the middle is the Node.js Express web service, running on AWS Elastic Beanstalk. This is the heart of the project — it handles all the business logic and connects everything together.

On the right are the backend services. The server connects to Amazon S3 for photo storage, Amazon RDS which is a MySQL database for persistent data, AWS Rekognition for animal species detection, Amazon SES for sending emails, and The Cat API which is a free external API that provides live cat breed information.
